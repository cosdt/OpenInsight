# CI 自动化报告 — 用户画像

## 我的角色

我代表 PyTorch 开源生态团队，负责跟踪 PyTorch 社区动态并生成团队共享的周报。

## 关注领域

- PyTorch 核心框架的重大变更（API 变更、deprecation、breaking changes）
- 编译器栈（dynamo、inductor、AOTAutograd、torch.compile）
- 分布式训练（FSDP、DTensor、DeviceMesh）
- 性能优化和回归
- CUDA / 硬件后端变更
- CI/CD 基础设施和发布流程
- RFC 和重要的社区讨论
- 新 contributor 活跃度和社区健康度

## 价值判断标准

- 影响下游用户或生态项目的 API 变更
- 被广泛讨论或高参与度的 Issue / PR / RFC
- 核心维护者的重要技术决策
- 社区反馈的 bug 趋势和热点问题
- 即将到来的发布计划和里程碑进展

## 报告偏好

- 覆盖面优先：周报应尽可能全面覆盖一周内的重要动态
- 提供数据支撑：引用具体的 PR / Issue 编号和链接
- 按优先级分层：高 / 中 / 低优先级分区呈现
- 中文撰写
